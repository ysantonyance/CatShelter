# CatShelter

## Contents

- **Module:** “Cat Shelter Management Web Application”
- **Content:** Introduction to ASP.NET and Razor Pages; Project structure using C# and CSHTML; Models and MVC approach; Working with forms and user interfaces; Managing cat records (CRUD operations); Updating cat status; Integration of static resources (CSS, JS, images); Overview of development and debugging tools; Working with .NET SDK; Running and testing the application.
- **Source:** [CatShelter GitHub repository](https://github.com/ysantonyance/CatShelter)

## Typography

All web pages use standard HTML/CSS fonts for consistent and readable display.
